# 3️⃣ I want to send this parcel, please

### Word meaning

* I → நான்
* want → வேண்டும்
* to send → அனுப்ப
* this → இந்த
* parcel → பார்சல்
* please → தயவு செய்து

### English (SVO)

```
Subject + Verb + Object
I want to send this parcel
```

### Tamil (SOV)

```
Subject + Object + Verb
```

👉 **நான் இந்த பார்சலை அனுப்ப வேண்டும்**

# 4️⃣ Do you want to send it by letter post or parcel post?

### Word meaning

* Do → (question marker)
* you → நீங்கள்
* want → விரும்புகிறீர்களா
* to send → அனுப்ப
* it → அதை
* by → மூலம்
* letter post → கடித அஞ்சல்
* or → அல்லது
* parcel post → பார்சல் அஞ்சல்

### English structure

```
Do + Subject + Verb + Object
```

### Tamil structure

```
Subject + Object + Verb
```

👉 **அதை letter post-ஆ அல்லது parcel post-ஆ அனுப்ப விரும்புகிறீர்களா?**

---
