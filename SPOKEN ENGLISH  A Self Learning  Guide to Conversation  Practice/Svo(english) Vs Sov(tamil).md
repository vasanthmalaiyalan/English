# 1️⃣ I want to send this parcel

### English Structure (SVO)

| Part    | Word         |
| ------- | ------------ |
| Subject | I            |
| Verb    | want to send |
| Object  | this parcel  |

Structure

```text
S + V + O
I want to send this parcel
```

### Tamil Structure (SOV)

| Part    | Word            |
| ------- | --------------- |
| Subject | நான்            |
| Object  | இந்த பார்சல்    |
| Verb    | அனுப்ப வேண்டும் |

Sentence

👉 **நான் இந்த பார்சலை அனுப்ப வேண்டும்**
