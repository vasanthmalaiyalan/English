நல்ல கேள்வி 👍
**தமிழ் ஏன் Subject + Object + Verb (SOV) order-ல் இருக்கிறது?**

இதற்கு முக்கிய காரணம் **மொழி அமைப்பு (language structure)**. உலக மொழிகள் எல்லாம் ஒரே மாதிரி இல்லை.

---

# 1️⃣ மொழிகள் இரண்டு முக்கிய sentence order-ல் இருக்கும்

உலகத்தில் பொதுவாக இரண்டு பெரிய sentence structures இருக்கிறது.

### 1. SVO (Subject Verb Object)

```text
Subject + Verb + Object
```

Example (English):

**I eat rice**

* I → Subject
* eat → Verb
* rice → Object

---

### 2. SOV (Subject Object Verb)

```text
Subject + Object + Verb
```

Example (Tamil):

**நான் சோறு சாப்பிடுகிறேன்**

* நான் → Subject
* சோறு → Object
* சாப்பிடுகிறேன் → Verb

---

# 2️⃣ தமிழ் ஒரு SOV language

தமிழ் மட்டும் அல்ல.

இந்த மொழிகளும் **SOV structure** பயன்படுத்தும்:

* தமிழ்
* தமிழ் குடும்ப மொழிகள் (Telugu, Kannada, Malayalam)
* Japanese
* Korean
* Hindi
* Turkish

Example:

**Hindi**

> Main chawal khata hoon
> நான் சோறு சாப்பிடுகிறேன்

Structure:

```text
Subject + Object + Verb
```

---

# 3️⃣ Verb கடைசியில் வருவதற்கு காரணம்

தமிழ் ஒரு **agglutinative language**.

அதாவது:

* verb-க்கு tense
* person
* number

எல்லாம் **verb-லேயே சேர்க்கப்படும்**.

Example:

**சாப்பிடுகிறேன்**

Breakdown:

* சாப்பிடு → eat
* கிற → present tense
* ஏன் → நான்

அதனால் **verb sentence முடிவில் வந்தால் meaning clear ஆகும்**.

---

# 4️⃣ English எப்படி வேறுபடுகிறது

English ஒரு **SVO language**.

Example:

**I eat rice**

English-ல் tense & person verb-ல் அதிகமாக attach ஆகாது.

அதனால் verb நடுவில் வரும்.

---

# 5️⃣ ஒரு நல்ல comparison
