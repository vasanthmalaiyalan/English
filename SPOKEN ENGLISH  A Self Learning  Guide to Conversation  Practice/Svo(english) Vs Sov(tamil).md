# 1️⃣ I want to send this parcel

### English Structure (SVO)

| Part    | Word         |
| ------- | ------------ |
| Subject | I            |
| Verb    | want to send |
| Object  | this parcel  |

Structure

```text
S + V + O
I want to send this parcel
```

### Tamil Structure (SOV)

| Part    | Word            |
| ------- | --------------- |
| Subject | நான்            |
| Object  | இந்த பார்சல்    |
| Verb    | அனுப்ப வேண்டும் |

Sentence

👉 **நான் இந்த பார்சலை அனுப்ப வேண்டும்**

# முக்கிய வேறுபாடு

### English

```text
Subject + Verb + Object
```

Example

```text
I eat rice
```

---

### Tamil

```text
Subject + Object + Verb
```

Example

```text
நான் சோறு சாப்பிடுகிறேன்
```

# Simple Trick

English sentence படிக்கும் போது:

1️⃣ Subject கண்டுபிடி
2️⃣ Verb கண்டுபிடி
3️⃣ Object கண்டுபிடி

பிறகு தமிழில் மாற்று:

```text
Subject + Object + Verb
```
